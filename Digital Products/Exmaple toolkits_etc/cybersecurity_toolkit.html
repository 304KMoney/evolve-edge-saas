Module 10
Cybersecurity Risk Assessment Scorecard
Rate your current security posture across 7 key domains (1 = Low Risk → 5 = High Risk)
📊

How to Score: Rate each security domain from 1 (excellent, low risk) to 5 (poor, high risk). A lower total score = stronger security posture. Use findings to prioritize improvements.

🔑 Password Security
1=Strong unique passwords + manager
5=Reused/weak passwords
📱 Multi-Factor Authentication
1=MFA on all critical accounts
5=No MFA in place
💾 Data Backup Protection
1=3-2-1 backup tested monthly
5=No backup system
💻 Device Security
1=Encrypted, updated, AV installed
5=Unpatched, no encryption
📧 Email & Phishing Protection
1=SPF/DKIM/DMARC + training done
5=No filtering or training
📡 Network Security
1=Secured router, VPN, firewall
5=Default settings, no VPN
☁️ Cloud Security
1=MFA, audited permissions, alerts
5=Shared logins, no monitoring
★ Score Interpretation Guide
🟢
7–14 Points
Strong Security Posture — Keep it up!
🟡
15–24 Points
Moderate Risk — Improvements needed
🔴
25–35 Points
High Risk — Immediate action required
